## Цель исследования
Сравнить качество решения прикладных задач горнодобывающей и нефтегазовой 
отрасли с помощью зарубежного (Gemini) и отечественного (YandexGPT) 
ИИ-ассистентов. Смоделировать сценарии использования чат-ботов для 
решения реальных производственных задач и оценить эффективность 
каждого инструмента.

## Выбранные кейсы

### Кейс 1: Диагностика неисправностей насосного оборудования
- **Описание:** Моделирование ситуации с неисправностью центробежного 
  маслонасоса НМ-1200 в гидросистеме буровой установки. 
  Симптомы: пониженное давление масла, повышенная температура корпуса, 
  вибрация выше нормы, пузырьки воздуха в масле.
- **Цель:** Проверить способность ботов к техническому анализу, 
  построению логики диагностики и объяснению физики процессов.
- **Запрос:** Пошаговый алгоритм диагностики в формате таблицы 
  с физическим обоснованием каждой причины.

### Кейс 2: Геологоразведка и интерпретация данных
- **Описание:** Предоставлены геохимические данные по четырём образцам 
  и описание геологического разреза (метаморфические сланцы, 
  гранитные интрузии, кварцевые жилы в зоне контакта).
- **Цель:** Оценить способность ботов к интерпретации геологических 
  данных, построению научной гипотезы и планированию 
  дальнейших работ.
- **Запрос:** Определить тип месторождения и предложить 
  детализированный план геологоразведочных работ.

## Методология
1. Для каждого кейса сформулирован единый запрос (промпт).
2. Запросы отправлены обоим ботам в новом чистом чате 
   для исключения влияния предыдущих диалогов.
3. Результаты сохранены в виде скриншотов в папку `results/`.
4. Проведён сравнительный анализ по шести критериям:
   - Логика и полнота ответа
   - Техническая грамотность
   - Глубина физического обоснования
   - Соответствие отраслевой специфике и нормативной базе
   - Наличие "галлюцинаций" (ложных фактов)
   - Удобство восприятия и следование формату
